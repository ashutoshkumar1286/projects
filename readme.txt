Below mention steps to run this program 

in command line
----------------
1.cd path to the directory that project is installed in
(ex:C:\Users\Admin>cd d:\fair-billing
    d:\fair-billing>java -jar session.jar input.txt)
Once program will executed sucessfully it will give the expected output.


In eclipse
----------------- 
import the project(directory name session) in eclipse and add the input.txt file as  commandline argument.

steps to set command line argument
1.Right Click on RunAs -> Runasconfiguration
2.Select Your main class(LogFileParser)
Click on the Arguments tab.set the path of input.txt file and save
3.Run as java application  it will give the expected output
